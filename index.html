🎉 이게 뭔데?

"촛불 끄셈 ㅋㅋ"은 생일 축하를 위한 인터랙티브 웹앱입니다. 이름과 나이를 입력하면 예쁜 생일 케이크가 만들어지고, 링크를 보내면 상대방이 실제로 마이크에 후~ 불어서 촛불을 끌 수 있어요!

앱 설치 없이 링크만 보내면 되니까, 카카오톡이나 문자로 간편하게 생일 축하를 전할 수 있습니다. 멀리 있는 친구한테도 케이크를 보낼 수 있는 거죠!

2026년 트렌드인 따뜻한 레트로 감성의 아날로그 디자인으로, 손으로 그린 듯한 스타일과 빈티지한 색감이 특징이에요.

🕯️ 🕯️ 🕯️

✨ 주요 기능

🎤

후~ 불어서 끄기

마이크에 실제로 바람을 불면 촛불이 하나씩 꺼집니다. 세게 불수록 빨리 꺼져요!

📱

자이로센서 반응

핸드폰을 기울이면 촛불의 불꽃이 실시간으로 흔들립니다. 기울기에 따라 연기도 같이 흘러가요!

🔗

링크 공유

앱 설치 없이 링크만 보내면 끝! 카카오톡, 문자, SNS 어디든 공유할 수 있어요.

🎨

나이별 테마

나이에 따라 케이크 디자인이 달라집니다. 아기부터 할아버지까지, 각 나이대에 맞는 특별한 케이크!

💌

비밀 메시지

촛불을 다 끄면 나타나는 비밀 메시지를 숨겨두세요. 특별한 한마디를 전할 수 있어요!

💨

B급 연기 효과

촛불을 끄면 초 개수에 따라 연기가 모락모락! 나이가 많으면 "화재 아닙니다" 경보가?!

🎈 🎈 🎈

📖 사용 방법

이름과 나이 입력

생일인 사람의 이름과 나이를 입력하세요. 비밀 메시지도 넣을 수 있어요! 메시지는 촛불을 다 끈 후에 공개됩니다.

케이크 미리보기

나이에 맞는 예쁜 케이크가 만들어집니다. 마음에 들면 링크를 복사해서 생일인 친구에게 보내세요!

링크 보내기

복사된 링크를 카카오톡, 문자, SNS 등으로 보내세요. 받는 사람은 별도 앱 설치 없이 바로 열 수 있습니다.

후~ 불어서 촛불 끄기

링크를 연 사람은 화면을 터치해서 불을 붙이고, 마이크에 후~ 불어서 촛불을 끕니다. 촛불이 다 꺼지면 축하 메시지와 함께 비밀 메시지가 공개돼요!

🎁 🎁 🎁

💬 자주 묻는 질문

무료인가요?

네, 완전 무료입니다! 회원가입도 필요 없어요.

앱을 설치해야 하나요?

아니요! 웹 브라우저에서 바로 사용할 수 있습니다. 링크만 열면 돼요.

마이크 권한은 왜 필요한가요?

촛불을 "후~" 불어서 끄려면 마이크가 필요해요. 마이크 데이터는 녹음되거나 저장되지 않고, 실시간으로만 사용됩니다.

마이크 없이도 사용할 수 있나요?

마이크 권한을 거부하면 촛불 끄기 체험은 어렵지만, 케이크를 보고 축하 메시지를 확인할 수는 있어요.

자이로센서가 안 되는데요?

데스크탑 브라우저에서는 자이로센서가 지원되지 않아요. 모바일(스마트폰)에서 사용하면 폰을 기울일 때 불꽃이 흔들리는 효과를 즐길 수 있습니다!

개인정보는 수집되나요?

아니요! 입력한 이름, 나이, 메시지는 URL에만 저장되고 서버에는 전송되지 않습니다. 자세한 내용은 개인정보처리방침을 확인하세요.

PC에서도 사용할 수 있나요?

네! 다만, 최고의 경험은 모바일에서 가능해요. 마이크로 불기, 자이로센서 기울이기, 진동 피드백 등 모바일 전용 기능이 있거든요.

케이크 링크는 얼마나 오래 유지되나요?

영원히요! 링크 자체에 모든 정보가 담겨있어서 서버 상관없이 계속 사용할 수 있어요.

🛠️ 만든 이야기

이 프로젝트는 "앱 설치 없이 생일 축하를 보내는 가장 간단한 방법"을 목표로 만들었습니다.

React 18과 Vite 5로 빌드했고, 서버 없이 GitHub Pages에서 무료로 호스팅하고 있어요. 모든 데이터는 URL 해시에 인코딩되어 있어서 서버가 필요 없는 완전한 서버리스 구조입니다.

Web Audio API로 마이크 입력을 실시간 분석하고, DeviceOrientation API로 자이로센서 데이터를 받아 불꽃과 연기의 움직임에 반영합니다.

2026년 디자인 트렌드인 "안티 AI 퍼펙션"을 반영해서, 일부러 손으로 그린 듯한 삐뚤빼뚤한 테두리, 아날로그 종이 질감, 빈티지 스탬프 스타일 등 따뜻한 레트로 감성을 담았습니다.

오픈소스 프로젝트로 GitHub에서 소스 코드를 확인하실 수 있습니다.
